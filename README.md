Pick the file that matches your computer, then follow the short install steps below. You do not need to be technical.

| Your computer | Download | What to do |
| --- | --- | --- |

## Install FounderOS

### 🍎 Mac (macOS 12.0 or newer)

1. Download the file for your Mac from the table above.
2. It lands in **Downloads**. Double-click the file (its name ends in `.dmg`).
3. A window opens showing the FounderOS icon and an Applications folder. Drag the FounderOS icon onto the Applications folder.
4. Open your Applications folder and double-click **FounderOS**.
5. Follow the setup screens.

The Mac version is signed by Apple, so it should open normally. If your Mac ever asks you to confirm, hold **Control**, click FounderOS, choose **Open**, then confirm **Open**.

### 🪟 Windows (Windows 10 or 11)

1. Download **FounderOS-win-x64.exe** from the table above.
2. It lands in **Downloads**. Double-click the file (its name ends in `.exe`).
3. You'll see a blue box that says **"Windows protected your PC."** This is normal for a brand-new app. It does **not** mean anything is wrong.
4. Click the small **More info** link inside that blue box.
5. A **Run anyway** button appears. Click it.
6. Windows may ask **"Do you want to allow this app to make changes to your device?"** Click **Yes**.
7. Follow the setup screens.

> **Why the blue box?** FounderOS is brand new and not yet registered with Microsoft, so Windows shows a caution the first time. It's safe — it comes straight from this official release page. **More info → Run anyway → Yes** is all you need. A signed Windows version is coming soon to remove this step.

## First Login

FounderOS creates a local admin account the first time it runs.

| Field | What to use |
| --- | --- |
| Username | `admin` |
| Password | The password shown during first launch |

Write the password down as soon as you see it. If you miss it, it's saved on your own computer — Mac: `~/.founderos/first-run-password.txt`, Windows: your user folder under `.founderos\first-run-password.txt`. You can change it in settings after you log in.
