# FounderOS Releases

Official public release feed for FounderOS installers, local server bundles, checksums, and update metadata.

Download the current approved release:

https://github.com/EverythingAI-Pro/founderos-releases/releases/tag/v0.4.2

## What To Download

- Apple Silicon Mac: `FounderOS-<version>-mac-arm64.dmg`
- Intel Mac: `FounderOS-<version>-mac-x64.dmg`
- Windows browser runtime: `founderos-server-<version>.zip`
- Mac/Linux browser runtime: `founderos-server-<version>.tar.gz`
- Checksums: `SHA256SUMS`

## Source Code Privacy

This public repository is only the release feed. It does not contain the private FounderOS product source code.

The private source repository is `EverythingAI-Pro/founderos-console`. Customers should not clone the source repo, run build commands, handle signing credentials, or use GitHub as part of normal setup.

## Updates

FounderOS uses this release feed for approved customer downloads and app updates. Customers install FounderOS from release assets; the product codebase stays private.
