# FounderOS — Releases

Official signed and notarized FounderOS desktop app releases.

Download the latest version from the [Releases](../../releases/latest) page.

- **Apple Silicon (M1/M2/M3/M4):** `FounderOS-<version>-mac-arm64.dmg`
- **Intel Mac:** `FounderOS-<version>-mac-x64.dmg`

Open the DMG and drag **FounderOS** into Applications. Signed with Apple Developer ID and notarized — no security warnings.

The app checks this repository for in-app updates automatically.
